# Lecturer Satisfaction Analysis

This project analyzes student satisfaction toward lecturers using dimensionality reduction techniques.

## 📊 Dataset
The dataset contains student evaluation scores of lecturers based on several criteria such as teaching quality, clarity, interaction, and overall satisfaction.

## ⚙️ Methods
- PCA (Principal Component Analysis)
- t-SNE (t-Distributed Stochastic Neighbor Embedding)

## 🛠 Tools
- Python
- Google Colab
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 🔍 Data Processing
- Data cleaning and preprocessing
- Feature selection
- Normalization of data
- Dimensionality reduction using PCA and t-SNE

## 📈 Visualization
- PCA scatter plot
- t-SNE clustering visualization

## 🎯 Objective
To identify patterns and insights from student evaluation data and understand key factors influencing lecturer performance.

## 📌 Results & Insights
- Identified clusters of lecturer performance based on student satisfaction
- Reduced high-dimensional data into meaningful visual patterns
- Provided insights for improving teaching quality

- Improve dataset size and quality
